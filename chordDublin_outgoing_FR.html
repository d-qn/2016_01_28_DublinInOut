Requérants d’asile renvoyés sur la base des accords Dublin (2009-2014)

Près de 85'000 requérants d’asile ont été transférés en six ans via les accords de Dublin. Les rubans montrent les flux bidirectionnels entre pays. A l’extrémité s’affiche le nombre total de transferts sortants pour chaque pays. Placez votre curseur ou tapotez sur une région du graphique pour voir le nombre exact de transferts.
